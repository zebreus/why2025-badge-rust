## Agent skills

### Issue tracker

Issues for this repo are tracked in GitHub Issues via the repo remote and the `gh` CLI. See `docs/agents/issue-tracker.md`.

### Triage labels

This repo uses the canonical triage labels `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, and `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

This repo is single-context: use the root domain context when present and treat `docs/adr/` as the canonical ADR location. See `docs/agents/domain.md`.
