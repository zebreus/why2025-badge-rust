== Mousefood Demo

Minimal example for using rust on the WHY2025 badge

This example builds as-is in this workspace. The badge-specific linker args come through `why2025-badge-app-no-std` plus a tiny root `build.rs` that calls `why2025_badge_build::configure`, so consumer apps only need the dedicated build helper and no checked-in `retain.txt`.

If you want to start a project, use [rust_starter](https://github.com/zebreus/why2025-rust-starter) instead, as that is standalone. See `Cargo.toml` and `build.rs` here for the integration shape used by the workspace examples.
