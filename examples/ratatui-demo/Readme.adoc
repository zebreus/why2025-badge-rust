== Ratatui Demo

Minimal example for using rust on the WHY2025 badge

This example builds as-is in this workspace. The badge-specific linker args now come through `why2025-badge-app-no-std` plus the tiny `src/build_script.rs` wrapper around `why2025_badge_app_no_std::configure_build`, so consumer apps no longer need a direct dependency on `why2025-badge-build` or a checked-in `retain.txt`.

If you want to start a project, use [rust_starter](https://github.com/zebreus/why2025-rust-starter) instead, as that is standalone. See `Cargo.toml` and `src/build_script.rs` here for the integration shape used by the workspace examples.
