= std hello world

Minimal Rust `std` App for the BadgeVMS std target.

This is the example exercised by the packaged-toolchain smoke checks.

Build with an installed `badgevms-std` toolchain:

----
rustup run badgevms-std cargo build --target riscv32imafc-unknown-badgevms --manifest-path examples/std-hello-world/Cargo.toml
----
