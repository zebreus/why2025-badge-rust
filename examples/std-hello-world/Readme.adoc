= std hello world

Minimal Rust `std` App for the BadgeVMS std target.

This is the example exercised by the rustup dist smoke checks.

Build with the installed BadgeVMS std toolchain:

----
cargo +nightly-2099-01-01 build --target riscv32imafc-unknown-badgevms --manifest-path examples/std-hello-world/Cargo.toml
----
