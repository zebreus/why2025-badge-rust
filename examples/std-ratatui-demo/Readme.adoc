= std ratatui demo

Richer Rust `std` App for the BadgeVMS std target using the existing Badge window plus the
mousefood embedded backend and ratatui widgets.

Build with the installed BadgeVMS std toolchain:

----
cargo +nightly-2099-01-01 build --target riscv32imafc-unknown-badgevms --manifest-path examples/std-ratatui-demo/Cargo.toml
----
